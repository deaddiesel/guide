§bInteractive Links
§7To create a clickable transition to another chapter or subchapter, use the syntax:
§6`[Button Text](file_name)`. §7IMPORTANT! No space is needed between the square bracket '§6]§7' and the parenthesis '§6(§7'.

§7Examples:
§r[Return to Introduction](introduction) §a(This works)
§c[Go to Demo Subchapter] (sub_demo1) §c(This does not work)

§7How it works:
§7• The text inside the square brackets §6[like this] §7will become clickable
§7• The text inside the parentheses §6(file) §7must match the actual name of the .md file (without extension)
§7• The link is automatically highlighted in §bcyan §7and underlined
§7• Upon clicking, the guidebook will instantly load the requested page
