§6Глава 14: Создание книги

§7В этой главе вы узнаете, как добавить собственный гайд в ваш мод, настроить его отображение в меню и скрыть руководство разработчика для игроков.

§l§6━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

§bСтруктура файлов

§7Для добавления книги создайте следующую структуру в ресурсах вашего мода:
§6`assets/<modid>/guide/`
• §6`book.json`§7 — конфигурация книги (название, иконка, настройки)
• §6`chapters/`§7 — папка с главами на ангнлийский (по умолчанию)
• §6`chapters/ru_ru/`§7 — папка с главами на русский (опционально)
• §6`chapters/ваш_язык/`§7 — перевод глав на ваш язык (опционально)

§l§6━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

§bНастройка book.json

§7Создайте файл §6`book.json`§7 рядом с папкой §6`chapters/`§7:
§6`{
§7  "name": "ваш_мод.book.guide",
§7  "namespace": "ваш_мод",
§7  "default_chapter": "introduction",
§7  "icon": "ваш_мод:textures/gui/book_icon.png",
§7  "dev_only": false
§6}`

§7Параметры:

• §e`name`§7 — ключ локализации для названия книги в меню (lang)
• §e`namespace`§7 — ID вашего мода (должен совпадать с папкой в assets)
• §e`default_chapter`§7 — имя файла первой главы (без .md)
• §e`icon`§7 — путь к иконке 16x16 в формате ResourceLocation
• §e`dev_only`§7 — если `§6true§7`, книга скрывается через конфиг

§l§6━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

§bЛокализация названия

§7Добавьте перевод ключа из §6`name`§7 в ваши файлы языка:
§6`assets/<modid>/lang/(ваш язык).json`
§6`{
§7  "ваш_мод.book.guide": "§6Ваш мод: Руководство§7"
§6}`

§7Игра автоматически подставит нужный язык при открытии меню выбора книг.

§l§6━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

§bСкрытие руководства разработчика

§7Если вы используете §6Guide§7 как библиотеку, ваше руководство разработчика по умолчанию скрыто для игроков.
§7Чтобы включить или выключить его, откройте конфиг:
§6`config/guide/guide-client.toml`
§7Найдите параметр:
§6`show_dev_manual = true`

§7Установите §6false§7, чтобы скрыть мануал из меню выбора. Разработчики модпаков могут изменить это значение в сборке.

§l§6━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

§bКак это работает:

§7• При запуске игры §6BookRegistry§7 автоматически сканирует §6assets/*/guide/book.json§7 всех модов
§7• Каждая книга изолирована по §6namespace§7 и использует свои ресурсы
§7• Конфиг §6show_dev_manual§7 фильтрует книги с флагом §6dev_only: true§7
§7• Меню выбора (§6BookSelectorScreen§7) отрисовывает список отфильтрованных книг с иконками
§7• При клике открывается §6GuideScreen§7 с переданным §6namespace§7, загружающий только нужные файлы

§l§6━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

§bСоветы

§7• Используйте иконки размером §616x16 пикселей§7 для корректного отображения в меню
§7• Не указывайте §6dev_only: true§7 для гайдов, которые должны видеть игроки
§7• Проверяйте пути к структурам и картинкам через §6namespace§7, чтобы избежать конфликтов
§7• Обновляйте §6guide-favorites.properties§7 только при необходимости — система управляет им автоматически
