§6Глава 13: Спойлеры и скрытый контент

§7В этой главе вы узнаете, как скрывать большие блоки текста, списки или лор за кликабельными заголовками.

§l§6━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

§bСинтаксис

§7Для создания спойлера используйте два тега:
§6`@spoiler:Заголовок`
... скрытый контент ...
§6`@endspoiler`

§7Параметры:
• §eЗаголовок§7 — текст, который виден всегда (кнопка)
• §eКонтент§7 — любой текст, картинки или теги внутри спойлера

§l§6━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

§bПримеры использования

§7§lСекретный лор (Скрыт):
§6`@spoiler:🔒 Секретная запись`

@spoiler:🔒 Секретная запись
§7Текст, который игрок увидит только после клика.
Здесь можно писать длинные истории, описания подземелий или скрытые рецепты.
@endspoiler

§7§lДлинный список изменений (Скрыт):
§6`@spoiler:📜 История версий`

@spoiler:📜 История версий
§7• §6v1.0§7 - Релиз
§7• §6v1.1§7 - Добавлены спойлеры
§7• §6v1.2§7 - Оптимизация Lazy Load
@endspoiler

§l§6━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

§bКак это работает:

§7• По умолчанию спойлер **свёрнут** (значок ▶)
§7• При клике он **раскрывается** (значок ▼) и показывает контент
§7• Высота страницы и скроллбар **автоматически пересчитываются**
§7• Спойлеры отлично экономят место в гайде, убирая визуальный шум
§7• Внутри спойлера можно использовать §6текст§7, §6картинки§7 и даже §6другие теги§7

§l§6━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

§bСоветы

§7• Используйте спойлеры для §6дополнительной информации§7, которая не нужна всем игрокам
§7• Не прячьте §6важные инструкции§7, которые нужны для прохождения
§7• Заголовки должны быть краткими и понятными
