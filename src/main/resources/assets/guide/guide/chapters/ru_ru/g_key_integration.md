§bБыстрый поиск для игроков

§7Больше не нужно вручную листать гайд в поисках нужного раздела. Движок книги позволяет мгновенно открывать статьи прямо из инвентаря:

§71. Откройте инвентарь, сундук или любой интерфейс крафта.
§72. Наведите курсор на интересующий предмет или блок.
§73. Зажмите клавишу **§6G§7** на **§a1,5 секунды§7**.
§74. В подсказке предмета появится анимированная оранжевая полоса сканирования.
§75. Как только загрузка достигнет **§a100%§7**, книга автоматически откроется на нужной странице!

§bИнструкция для создателей гайдов

§7Чтобы привязать предметы к создаваемой главе, добавьте специальный тег **@bind:** в самую первую строку вашего `.md` файла.

§7Пример синтаксиса тега привязки:
§6`@bind:minecraft:furnace,minecraft:crafting_table`

§7Вы можете указать любое количество ID предметов через запятую. Если предмет привязан к нескольким главам, боковое меню книги автоматически отфильтрует и покажет игроку только эти статьи, скрыв всё остальное!
