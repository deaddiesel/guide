§7Для сборки некоторых сложных конструкций вам потребуются базовые компоненты. В списке ниже представлены ключевые ресурсы, которые необходимо подготовить заранее:

§l§6━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

§bСетка верстака (3х3)

§7Ниже приведена точная схема расположения элементов в сетке верстака для создания базового энергетического блока:

@matrix_craft:minecraft:furnace, minecraft:cobblestone, minecraft:cobblestone, minecraft:cobblestone, minecraft:cobblestone, minecraft:redstone, minecraft:cobblestone, minecraft:cobblestone, minecraft:cobblestone, minecraft:cobblestone,center

§7Повторите указанную раскладку в обычном верстаке, чтобы получить готовый предмет, указанный в слоте вывода справа.

§l§6━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

§bВыравнивание сетки крафта

§7Как и изображения, сетку крафта можно выравнивать по горизонтали. Для этого добавьте параметр в конец директивы:

§7§lПо центру:
§6`@matrix_craft:результат, слот1, ..., слот9, center`

§7§lПо левому краю:
§6`@matrix_craft:результат, слот1, ..., слот9, left`

§7§lПо правому краю:
§6`@matrix_craft:результат, слот1, ..., слот9, right`

§7Параметр выравнивания всегда указывается **§6последним§7** через запятую после всех 9 слотов крафта.
